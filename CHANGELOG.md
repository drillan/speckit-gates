### Added

- Initial implementation of speckit-gates quality gate skills package
- `planning-validate` skill - Validates planning artifacts after `/speckit.plan`
- `implementation-verify` skill - Verifies implementation coverage after `/speckit.implement`
- `docs-sync` skill - Synchronizes documentation after `/speckit.implement`
- `progress-report` skill - Displays progress dashboard on demand
- `release-check` skill - Validates release readiness on demand
- Shared infrastructure for path resolution and output formatting
- README.md with installation and usage instructions
